# easy-front-vue-cli3

基于 vue cli3 的项目工程模板 2.0，详细的使用手册请查阅[easy-front-vue-cli3 使用手册](https://www.yuque.com/u40512/in31e0/lnpqim)

## 使用方法

### 安装依赖

```shell
npm install
```

### 调试模式

```shell
npm run dev
```

### 打包生产环境

```shell
npm run build
```

### 打包测试环境

```shell
npm run test
```

### 打包预发布环境

```shell
npm run pre-release
```

### 检查语法和修复文件

```shell
npm run lint
```
